Granular Database Tables & Logs

1. Registration Data Registry

Full list of retailers and mechanics with user ID, name, email, firm name, mobile, points balances, approval statuses, and registration timestamps.

2. Product Details Catalog

Product master table including product ID, catalog name, pack size, barcode QR codes, dealer prices, and tier-specific point multiplier configurations.

3. Purchase Requests Ledger

Chronological records of all submitted point scan claims. Tracks user IDs, scan dates, quantities, estimated amounts, and pending/approved/rejected states.

4. Purchase History Log

Secure record of approved transactions. Includes Purchase IDs, verified points awarded, authorized billing amounts, and audit signatures of the approving agent.

5. Points Ledger Statements

Comprehensive audit ledger mapping point credit scans, point debit redemptions, running point balances for every user, and official validation remarks.

6. Admin Users Directory

Organizational list of systems administrators and executive owners. Contains email, password, mobile, approval role, and active status indicators.

7. OTP Authentication Logs

Security logs of 2FA OTP codes issued to users during logins. Tracks generation timestamps, code values, validation outcomes, and usage timestamps.

8. System Activity Logs

Low-level system activity records: timestamps, user profiles, actions (e.g. login, scan, catalog change), functional modules, and remarks.
